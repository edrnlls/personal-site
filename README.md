# Personal Site — Eduardo Dornelles

Live: https://personal-site-edrnlls.vercel.app/

My personal landing page. Built as the first project of my journey from franchise operations background into Full Stack Product Engineering with Applied AI.

## Stack

- Vanilla HTML + CSS (no framework — kept intentionally simple as the first deploy)
- Deployed on [Vercel](https://vercel.com)
- Source hosted on GitHub

## Why this project

This is Boss Fight #1 of a personal 24-week training path toward international remote contracting as a Product Engineer. The goal of this specific project wasn't the site itself — it was to complete the full deployment cycle for the first time: local code → Git → GitHub → Vercel → live URL.
